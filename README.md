📘 宿泊施設予約管理システム（team003）
📌 概要
本プロジェクトは、鳥取県内の小規模宿泊施設を対象とした 宿泊予約管理システム です。
従来の電話・メールによる手作業の予約管理から脱却し、空室確認〜予約完了までをオンラインで完結 できる仕組みを提供します。
観光需要の増加に伴い、対応漏れ・二重予約・予約確定までの時間が長いといった課題を解決することを目的としています。
